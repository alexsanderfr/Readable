# Readable
Blog web app

### Prerequisites

* [Node.js](https://nodejs.org)
* [npm](https://www.npmjs.com/)

### Running app

* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`
* Install and start the app
    - `cd frontend`
    - `npm install`
    - `npm start`
